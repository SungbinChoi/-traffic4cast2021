
# How to start to train models

1. Prepare train data
  Unzip all data zip files ('BERLIN' 'CHICAGO' 'ISTANBUL' 'MELBOURNE','NEWYORK', 'VIENNA'), 
   move them to '$BASE_DIR/0_data' folder
    e.g., '$BASE_DIR/0_data/BERLIN'

2. Start training model
  e.g., 
    cd $BASE_DIR/train/t1m1
    python run.py
